## Goal

Introduce **Universe** as the top-level context for game content.

Universe acts as a tenant-like container for a specific world or game configuration.
All Codex entities such as cards, heroes, and elements belong to a Universe.

## Scope

This milestone includes:

- introducing the **Universe** entity
- adding CRUD for Universe
- adding Universe routes in admin UI
- making Universe the root context for future Codex sections
- preparing the system for multi-world content isolation
