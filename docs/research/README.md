# Research

Point-in-time investigations that inform design decisions — comparative studies, stress tests of models against existing systems, technical surveys. Separate from [Design](../design/README.md) docs, which describe what the system _is_; research describes what _informed_ choices.

Research docs are snapshots: they capture the state of thinking at the time and are not expected to stay current as the system evolves.
